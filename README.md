> 根據內政部消防署的統計，在104年~108年年發生水域事故的人數逐年攀升。本團隊整合政府開放的Open Data以及大鵬灣的水上活動規範，開發出APP以及網頁雙平台，讓使用者隨時都可以查詢水域的即時狀況，並透過APP即時大聲響鈴警示民眾該區域之海域安全性，以減少憾事的發生。
